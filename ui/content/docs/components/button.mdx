### Examples

import { ArrowRight, LogIn } from "lucide-react";

#### Text & icon

A button with icon beside the text content. Use `lucide-react` icon is recommended.

<div className="flex gap-4 h-64 border items-center justify-center rounded-xl">
  <Button>
    Register <ArrowRight />
  </Button>
  <Button variant="outline">
    Login <LogIn />
  </Button>
</div>

```jsx
<Button>Register <ArrowRight /></Button>
<Button variant="outline">Login <LogIn /></Button>
```

#### Variants

Control the variant either `default`, `outline`, `destructive`, `dark`. `default` is the default value.

<div className="flex gap-4 h-64 border items-center justify-center rounded-xl">
  <Button>Default</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="destructive">Destructive</Button>
  <Button variant="dark">Dark</Button>
</div>

```jsx
<Button>Default</Button>
<Button variant="outline">Outline</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="dark">Dark</Button>
```

#### Style

Control the style either `default`, `icon`, `shadow`. `shadow` is the default value.

<div className="flex gap-4 h-64 border items-center justify-center rounded-xl">
  <Button>Shadow</Button>
  <Button style="default">Default</Button>
  <Button style="icon">
    <LogIn />
  </Button>
</div>

```jsx
<Button>Shadow</Button>
<Button style="default">Default</Button>
<Button style="icon"><LogIn /></Button>
```
